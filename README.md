Project Author: Espi Madrazo

Project Description: Framework for APIs tests automation and basic requests tests. The public API from "https://jsonplaceholder.typicode.com" was used.

*Required dependencies*
- pip (included in Python if downloaded from python.org)
- pytest: pip install pytest
- pytest-html: pip install pytest-html
- pytest-check: pip install pytest-check
- requests: pip install requests

*Instructions for tests execution*
(Basic pytest configuration has been set up in pytest.ini)

Terminal commands:
- Show help on command line and config file options: pytest -h
- Run all tests (included in tests directory): pytest
- Run all tests in a module: pytest test_mod.py

For further commands check the official documentation: "https://docs.pytest.org".
